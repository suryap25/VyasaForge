# VyasaForge Branding

## Product Name

VyasaForge

## Tagline

Forge structured knowledge into publishable documents.

## Motto

Plan. Write. Review. Publish.

## CLI Identifier

`vf`

The full CLI name is also supported:

```powershell
vyasaforge --help
```

## Name Meaning

`Vyasa` refers to the arranger, compiler, and organizer of knowledge. In this
project, it represents the act of turning scattered ideas, outlines, and expert
input into structured knowledge.

`Forge` represents transformation: raw ideas become shaped, reviewed,
validated, and publishable artifacts.

Together, `VyasaForge` means a production pipeline for transforming knowledge
into finished documents.

## Naming Conventions

- Product name: `VyasaForge`
- CLI: `vf`
- Full CLI: `vyasaforge`
- Workspaces: lowercase slug names such as `aws-cloud-security`
- Generated documents: `<workspace-name>.docx` or `<workspace-name>.pdf`
- Example use cases: technical handbooks, architecture guides, runbooks, playbooks, PRDs, and training material

Future documentation should avoid making any one domain look like the product
boundary.
